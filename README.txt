# E Commerce Shop

## Introduction

The task of the application is to demonstrate a simple E Commerce application, where the front-end website programmed in the Angular framework connects to the Spring Boot back-end.

## Technologies

- Spring Boot
- Angular
- MySQL
- Docker

## Wymagania

Requirements, np.:

- Amazon Coretto 17
- Node.js v18
- MySQL 8.0
- Docker
- Docker Compose
